# Cline: Интерфейс и настройки

## Режимы работы

### ACT Mode (по умолчанию)
- Полный доступ ко всем инструментам
- Может читать, писать, выполнять команды
- Для реализации и изменений

### PLAN Mode
- Только обсуждение и планирование
- НЕ может писать файлы или выполнять команды
- Для безопасного обсуждения архитектуры

**Переключение:** кнопка Plan/Act внизу экрана (только пользователь)

---

## Thinking Budget

| Параметр | Описание |
|----------|----------|
| Thinking Budget | Лимит токенов на "размышления" модели |
| GLM-4 | 6,000 токенов |
| DeepSeek R1 | до 64,000 токенов |

Модель "думает" перед ответом. Больше budget = глубже анализ, но медленнее.

---

## Auto-Approve

| Уровень | Что одобряется | Риск |
|---------|----------------|------|
| Off | Ничего | Нет |
| Read-only | Чтение файлов | Нет |
| Write | Чтение + запись | Средний |
| Execute | + команды | Высокий |
| Full | Всё | Очень высокий |

**Рекомендация:** Read-only или Write (с Git)

---

## Enable Notifications

| ON | OFF |
|----|-----|
| Всплывающие уведомления | Тихая работа |
| "File created" | Без визуального шума |

НЕ влияет на подтверждение действий — это Auto-Approve.

## Хранение контекста

```
C:\Users\{user}\.cline\data\
├── tasks/{task_id}/
│   ├── api_conversation_history.json  # История API
│   ├── ui_messages.json               # Сообщения UI
│   └── task_metadata.json             # Метаданные
├── workspaces/{hash}/
│   └── workspaceState.json            # Состояние workspace
└── globalState.json                   # Глобальное состояние
```

**Context Window:** 204,800 токенов максимум

| **GLM-4** | Русский, баланс скорости | Thinking budget 6K |
