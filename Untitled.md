### 2. Project Manager (`servers/project_manager/`)

- **Назначение:** Управление проектами ПЗ
- **Инструменты:**
    - `init_project` — инициализация нового проекта
    - `detect_project` — определение типа проекта
    - `load_project` — загрузка существующего проекта
    - `get_project_info` — информация о проекте
    - `add_pdf` — добавление PDF в проект
    - `get_state` / `update_state` — работа с состоянием
