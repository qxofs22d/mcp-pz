# Требования к окружению

- Python 3.14.2 (уже установлен в виртуальном окружении).
- Виртуальное окружение: `d:\dev\.venvs\aose` (создано пользователем). Активируй его перед установкой пакетов: .\venv\Scripts\activate
- После активации установи базовые пакеты (они будут добавляться по мере необходимости). Начни с: pip install mcp # или mcp-python-sdk
- Для каждого модуля будут свои зависимости (см. спецификации). Общий файл `requirements.txt` будет пополняться.

**Примечание:** Для работы с PDF на Windows могут потребоваться дополнительные системные библиотеки (например, для pdfplumber). Если возникнут ошибки, сообщи пользователю.
