# Чек-лист для поэтапной проверки

## Менеджер проекта
- [ ] Создание проекта с папками pdf, data, output и state.json.
- [ ] Добавление PDF копирует файл и обновляет state.
- [ ] Загрузка проекта восстанавливает состояние.
- [ ] Сохранение текста раздела создаёт файл и обновляет state.

## Data Core
- [ ] Создание и изменение параметров (set_parameter, get_parameter).
- [ ] Добавление и вычисление формул (add_formula, execute_formula).
- [ ] Запуск внешнего Python-скрипта (run_script) с обновлением данных.
- [ ] Экспорт/импорт данных (export_data, import_data).

## Векторное хранилище
- [ ] Индексация текстового файла (add_document_chunks).
- [ ] Поиск по запросу возвращает релевантные чанки (search).
- [ ] Фильтрация по документам.

## Декомпозиция PDF
- [ ] Обработка PDF с оглавлением даёт правильную иерархию разделов.
- [ ] Таблицы извлекаются в структурированном виде.
- [ ] Формулы (содержащие "=") идентифицируются.
- [ ] Названия чертежей находятся.

## Интеграция
- [ ] Все серверы запускаются одновременно (скрипт запуска).
- [ ] Cline может вызывать инструменты всех серверов.
- [ ] Полный цикл: создание проекта → загрузка PDF → декомпозиция → индексация → ввод данных → генерация текста → сборка документа.
