# Спецификация модуля: Logger

## Назначение
Запись сообщений диалога в JSONL-файл для последующего анализа и индексации.

## MCP-инструменты
- `log(role, type, content, metadata=None)` — записывает сообщение в файл `./.dialogue/dialogue.jsonl`. Поля:
  - `timestamp` (ISO формат) — добавляется автоматически.
  - `role` — "user" или "assistant".
  - `type` — строка, например "regular", "expert", "system".
  - `content` — текст сообщения.
  - `metadata` — объект JSON (опционально).

## Внутренняя структура
- При запуске создаёт папку `.dialogue`, если её нет.
- Открывает файл `dialogue.jsonl` в режиме append.
- При получении запроса дописывает строку.

## Зависимости
- Стандартные: json, datetime, os, pathlib.
- `mcp` для обёртки.

## Критерии готовности
- Сервер запускается, принимает запросы и записывает их в файл.
